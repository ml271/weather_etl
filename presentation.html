CAPABILITIES

What the platform delivers

🔄
AUTOMATED HOURLY ETL
Airflow DAG runs every hour, fetching fresh 7-day forecasts. 3 retries with backoff. Fully idempotent — reruns never corrupt data.
🌍
ANY CITY ON DEMAND
Search any city globally. If no data exists, the backend fetches and stores it immediately — no Airflow run required.
⚠️
CONFIGURABLE ALERTS
Rule engine driven by alerts_config.yaml. 7 default rules: extreme heat, frost, storm, heavy rain, UV warning, heatwave, snowfall.
📈
INTERACTIVE DASHBOARD
7-day strip, active alerts panel, 24/48/72/96h hourly chart. Click any forecast day for a detailed per-day breakdown chart.
